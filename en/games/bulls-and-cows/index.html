How to Play

  • The system generates a number with N digits (3-6).
  • Allow repeats determines whether digits can repeat.
  • Submit a number with the selected length.
  • A = correct digit and correct position
  • B = correct digit but wrong position
  • Each submission appears in the history on the left
  • The game records fastest time and fewest guesses
