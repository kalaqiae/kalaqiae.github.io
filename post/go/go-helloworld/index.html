下载Go

下载地址： https://golang.google.cn/dl/

安装

安装完成后会自动配置环境GOPATH，在cmd中输入go version测试，显示版本号则安装成功

HelloWorld Demo

新建hello.go文件，复制保存以下代码到文件中，使用go run运行

package main

import "fmt"

func main() {
   fmt.Println("Hello, World!")
}

例如在F:\go文件夹下新建hello.go，写好代码保存后，在cmd中输入 go run F:\go\hello.go

运行起来速度可能会比较慢，等个几秒
